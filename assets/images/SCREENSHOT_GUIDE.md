# Screenshot Guide (Widget Tree Assignment)

Capture and save these two screenshots after running:

```bash
flutter run --target=lib/main_widget_tree.dart
```

Required files:

- `widget_tree_initial.png` → app just launched
- `widget_tree_updated.png` → after an interaction (for example: increment counter + toggle theme)

Save both files in this folder (`assets/images/`) so they render in `README.md`.
